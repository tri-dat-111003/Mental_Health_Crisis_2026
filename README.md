# 🌍 Global Mental Health Crisis Index 2026 Analysis

An interactive data dashboard built with **Streamlit** to analyze the relationships between mental health challenges, economic status, and technological infrastructure across the globe.

🚀 **Live Demo:** [Link](https://mental-health-analysis-2026.streamlit.app/)

---

## 📊 Project Overview
This project provides a comprehensive look into the **Global Mental Health Crisis Index 2026** dataset. The goal is to identify how factors like **Internet Penetration**, **Social Media Usage**, and **National Income** correlate with mental health indicators such as the Crisis Index and Treatment Gaps.

As a Data Analyst, I developed this tool to transform raw Kaggle data into actionable visual insights that can help stakeholders understand regional disparities in mental health investment.

## 🛠 Tech Stack
- **Language:** Python 3.14
- **Framework:** Streamlit (Web Interface)
- **Data Manipulation:** Pandas, NumPy
- **Visualization:** Plotly Express (Interactive Charts), Seaborn, Matplotlib
- **Statistical Analysis:** Statsmodels (OLS Trendlines)

## 🌟 Key Features
- **Interactive Filtering:** Drill down by **Region**, **Income Group**, and **Crisis Severity**.
- **Correlation Mapping:** Analyzed the impact of digital connectivity (Internet %) on the mental health investment gap.
- **Social Media Risk Analysis:** Visualized the relationship between social media time and anxiety levels.
- **Responsive Design:** Optimized layout for wide-screen data exploration.

## 🚀 How to Run Locally

  **1. Clone the repository:**
  
  ```Bash
  git clone https://github.com/tri-dat-111003/mental-health-crisis-2026.git
  cd mental-health-crisis-2026
  ```
  **2. Create a virtual environment:**
  
  ```Bash
  python -m venv venv
  source venv/bin/activate  # On Windows: venv\Scripts\activate
  ```
  **3. Install dependencies:**
  
  ```Bash
  pip install -r requirements.txt
  ```
  **4. Launch the dashboard:**
  
  ```Bash
  streamlit run app.py
  ```
